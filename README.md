Sistem Optimasi Menu MBG (Metode Big M)
Aplikasi berbasis web untuk menghitung optimasi porsi menu MBG (Makan Bergizi Gratis) berdasarkan anggaran dan standar gizi menggunakan algoritma Big M (Simplex)
