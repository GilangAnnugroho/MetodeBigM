Sistem Optimasi Menu MBG

Optimasi porsi menu MBG berdasarkan anggaran, jumlah siswa, kebutuhan gizi minimal, serta berat porsi menggunakan Metode Big M (Simplex).

Input Lengkap

Masukkan anggaran, jumlah siswa, biaya per porsi, berat per porsi, dan kandungan gizi tiap menu.

RHS Otomatis

Nilai batas (RHS) dihitung otomatis dari standar kebutuhan per siswa Ã— jumlah siswa, sehingga mengurangi kesalahan input.

Perbandingan Hasil

Sistem menampilkan baseline standar gizi per siswa (sebelum optimasi) dan gizi per siswa setelah optimasi, serta berat porsi akhir yang diterima setiap siswa.

Alur Singkat Sistem
  1. Input anggaran maksimal dan jumlah siswa.
  2. Input menu (nama, biaya, berat per porsi).
  3. Input kandungan gizi per porsi dan standar kebutuhan gizi per siswa.
  4. RHS dihitung otomatis dan sistem melakukan optimasi Big M + Simplex.
  5. Hasil menampilkan porsi, gizi per siswa, dan berat porsi akhir per siswa.
